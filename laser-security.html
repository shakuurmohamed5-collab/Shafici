Overview

A hardware-based home security system designed to detect unauthorized entry using a laser and light sensor mechanism. When the laser beam is interrupted, the system triggers an alarm, providing a simple and cost-effective intrusion detection solution.

My Role

  • team lead and project manager
  • Designed the system logic and detection mechanism
  • Selected and integrated electronic components
  • Built and tested the full hardware prototype
  • Documented the project as a final-year thesis

How the System Works

A laser beam is continuously directed toward an LDR sensor. Under normal conditions, the laser remains aligned with the sensor. When the beam is interrupted, the change in light intensity is detected by the circuit, activating a buzzer and visual indicators to signal a security breach.
