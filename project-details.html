Enterprise SOC & Network Security Lab

FortiGate • Wazuh • SIEM • SOC Monitoring

Overview

This project demonstrates a hands-on enterprise SOC environment where firewall traffic, system logs, and security alerts are monitored and analyzed using FortiGate and Wazuh. The lab simulates real-world segmentation, logging, and attack detection.

What I Did

  • Designed a segmented enterprise network (WAN, LAN, Servers)
  • Configured FortiGate firewall, NAT, and security policies
  • Integrated firewall and host logs into Wazuh SIEM
  • Deployed Wazuh agents on Linux and Windows systems
  • Simulated SSH brute-force attacks and validated detections
